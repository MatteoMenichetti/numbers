This project can be use for generate the primes number between 2 and the input selected.

The input number must be major of 2 for a simple reason.

The makefile included can be used for compile the source code on local machine (utility make is required). After that
you are navigate on directory "numbers", you can use the utility make with this order:

1) make
2) make install
3) make clean

Attention: with make clean you delete each file .o and binary, so this action must made only after the execution.

The execution can be started with ./main (number).

PS: the included makefile is created for UNIX-like OS.
